# Agent Templates & Patterns

## Minimum Viable Agent
```
SOUL.md (≤1,500 chars) — identity + rules + boundaries
AGENTS.md — operating procedures
IDENTITY.md — name, emoji, vibe
```

## Production Agent (adds)
```
USER.md — human context
TOOLS.md — environment notes
MEMORY.md — long-term memory index
memory/ — extended memory files
skills/ — custom skills if needed
```

## Tool Access Tiers
- **Minimal:** profile: "coding", deny most
- **Standard:** profile: "coding", allow: ["group:web"]
- **Communicator:** + allow: ["group:sessions"]
- **Full:** profile: "full" (requires justification)
